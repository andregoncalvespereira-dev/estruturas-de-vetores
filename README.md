# Trabalhando com vetores em c++

**Nesse repositor, está presente alguns exercicios feito em sala de aula que envolve conseito de vetorer usando numeros inteiros e caracteres e tambem:**

*Variaveis;
*Tipos de variaveis;
*Presentção de informação no terminal;
*Entrada de infomação;
*Saida de informação;
*Declaração de vetores e acesso dos seus elementos;

# :wink: como executa 
* Em nossa aula, usamos a ferramenta online Online c++ compliler, presente no link:
<a href="https://www.onlinegdb.com/online_c++_compiler"> Online c++ compiler </a>

*Basta colar o codigo e apertar o botão Rum, um botão verde que fica na parte superior direito da tela.
